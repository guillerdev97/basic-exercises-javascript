Basic Exercises JavaScript

Exercise01. Write "Hello World!".

Exercise02. Create a function to sum 3 + 5 and show the result.

Exercise03. Create a class to sum, subtract, multiply and divide. In this case 3 + 5 and 3 * 5.

Exercise04. Write a program that asks the user name and returns a welcome text with the user's name (by prompt).

Exercise05. Write a program that asks the user name and returns a welcome text with the user's name (by input).


Exercise06. Write a program that asks the user two numbers (0-10) by input and show the result of sum the two numbers.
